Mode klasik

Battle Royale

Alur untuk turnamen BR murni: bracket tim tetap, babak eliminasi, dan fokus ke poin matchday tanpa lapisan kelas tambahan.

  • Bracket 12 atau 15 tim siap pakai
  • Input hasil babak & agregasi poin
  • Ringkasan posisi untuk pengumuman cepat
  • Cocok untuk event satu format BR end-to-end

Mode squad terstruktur

Class squad

Untuk fast tournament dengan pembagian kelas (squad): kontrol slot per kelas, jadwal lintas meja, dan poin yang tetap konsisten antar babak.

  • Slot & label per kelas squad
  • Poin mengikuti aturan class yang kamu tetapkan
  • Klasemen per kelas + gabungan opsional
  • Ideal untuk komunitas dengan banyak sub-tim
